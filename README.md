# Excel Assignment (Аналитик)

## Описание
Выполнено тестовое задание на позицию аналитика.

## Что сделано:
- Объединение данных из разных листов
- Построение сводной таблицы
- Расчет доли (% от общего)
- Сегментация по общему трафику
-  A/B тест (Test / Control / Out)
- Добавлен (срез)

## Инструменты:
- Excel (сводные таблицы, формулы, сегментация)
